## Tipos de inferĂȘncia

Estes tipos transformam uma string de template em tipos TypeScript concretos em tempo de compilaĂ§ĂŁo.

### `ParsePath<S>`

O tipo principal. Dado um template literal, infere `params`, `searchParams` e `fragment`:

```typescript

type Route = ParsePath<'/users/:id.number?page.number&active.boolean#section'>
// {
//   params: { id: number }
//   searchParams: { page: number, active: boolean }
//   fragment: { section: string }
// }
```

### `ParseParams<S>`

Infere apenas os parĂąmetros de rota:

```typescript

type Params = ParseParams<'id.number/posts/[slug]/tags/[...items]'>
// { id: number, slug: string, items: string[] }
```

Suporta todas as sintaxes: `:param`, `[param]`, `[~param]`, `[...slug]`, e `.type`.

### `ParseSearch<S>`

Infere os parĂąmetros de consulta:

```typescript

type Search = ParseSearch<'q&page.number&tags.array'>
// { q: string, page: number, tags: string[] }
```

### `ParseFragment<S>`

Infere o fragmento:

```typescript
import type { ParseFragment } from 'url-ast'

type Frag = ParseFragment<'section'>
// { section: string }
```

### `TypeMap`

Mapeia as strings de tipo (`'number'`, `'boolean'`, `'string'`, listas com vĂ­rgula) para os tipos JavaScript correspondentes:

```typescript

  number:  number
  boolean: boolean
  string:  string

```
