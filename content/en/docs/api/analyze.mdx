# Reference: Analyze

The `Analyze` class is url-ast's **high-level API**. It receives a URL (or template), builds the AST internally, and exposes methods to extract parameters, query, fragment, origin, and route metadata.

## Constructor

```typescript
new Analyze(input: string)
new Analyze(input: string, base: AnalyzeChain)
new Analyze(input: string, options: AnalyzeNewOptions)
```

| Parameter | Type | Description |
| :--- | :--- | :--- |
| `input` | `string` | The URL or template to parse |
| `base` | `AnalyzeChain` | Template for type casting and matching |
| `options` | `AnalyzeNewOptions` | Object with `{ parser?: AnalyzeChain }` |

---

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `ast` | `AST<string>` | The syntax tree generated by the parser |
| `errors` | `ErrorLog[]` | Parse errors found |

---

## Data extraction methods

### `getParams()`

Extracts route parameters (path params).

```typescript
getParams(): AnalyzeParamsResult
```

- **Without template:** returns `string[]` with declared parameter names.
- **With template:** returns a typed object with converted values.

```typescript
const template = new Analyze('/users/:id.number/:name')
const parsed = new Analyze('/users/42/john', template)

parsed.getParams() // { id: 42, name: 'john' }
```

### `getSearchParams()`

Extracts query parameters.

```typescript
getSearchParams(): AnalyzeSearchParamsResult
```

- **Without template:** returns `Map<string, string | string[]>`.
- **With template:** returns a typed object.

```typescript
const template = new Analyze('/search?q&page.number&tags.array')
const parsed = new Analyze('/search?q=ts&page=2&tags=web,api', template)

parsed.getSearchParams()
// { q: 'ts', page: 2, tags: ['web', 'api'] }
```

### `getFragment()`

Extracts the URL fragment (hash).

```typescript
getFragment(): AnalyzeFragmentResult
```

```typescript
const template = new Analyze('/page#section')
const parsed = new Analyze('/page#intro', template)

parsed.getFragment() // { section: 'intro' }
```

### `getStaticProps()`

Returns properties from static segments of bracket and catch-all routes.

```typescript
getStaticProps(): Record<string, string | string[]>
```

```typescript
const template = new Analyze('/docs/[...path]')
const parsed = new Analyze('/docs/api/reference/analyze', template)

parsed.getStaticProps() // { path: ['api', 'reference', 'analyze'] }
```

## Origin methods

### `getPathname()`

Returns the URL path, without origin, query, or fragment.

```typescript
getPathname(): string
```

### `getProtocol()`

```typescript
getProtocol(): string | undefined
```

### `getHostname()`

```typescript
getHostname(): string | undefined
```

### `getPort()`

```typescript
getPort(): string | undefined
```

```typescript
const p = new Analyze('https://api.example.com:8080/v1/users')
p.getProtocol() // 'https'
p.getHostname() // 'api.example.com'
p.getPort()     // '8080'
p.getPathname() // '/v1/users'
```

## Template and parsing

### `setParser(base)`

Associates a template with an existing instance. Returns a new typed instance.

```typescript
setParser<S extends AnalyzeChain>(base: S): AnalyzeInstance<string, S>
```

```typescript
const url = new Analyze('/users/42')
const template = new Analyze('/users/:id.number')
const parsed = url.setParser(template)

parsed.getParams() // { id: 42 }
```

## Serialization

### `getBuffer()`

Serializes the instance to a binary `Buffer`.

```typescript
getBuffer(): Buffer
```

### `Analyze.fromBuffer(buffer)`

Restores an instance from a serialized buffer.

```typescript
static fromBuffer(buffer: Buffer): AnalyzeChain
```

### `Analyze.create(input, base?)`

Static factory with full type inference.

```typescript
static create<P extends string>(input: P): AnalyzeInstance<P, undefined>
static create<P extends string, B extends AnalyzeChain>(input: P, base: B): AnalyzeInstance<P, B>
```

```typescript
const a = Analyze.create('/path/:x.string')
const buf = a.getBuffer()
const restored = Analyze.fromBuffer(buf)
```
